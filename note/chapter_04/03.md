# deno平台API

## API 使用方式

- `Deno.platform` 会输出一个对象 `{arch, os}`
  - `arch {string}` 操作系统的CPU 架构，例如 x64是64位
  - `os {string}` 操作系统的平台名称
    - mac: 苹果Mac OS系统
    - win: Windows 系统
    - linux: Linux 系统

```js

// 环境变量信息
const _env = Deno.env();
console.log(_env);
// Linux export: { HOME:'...', PWD:'...', SHELL:'...',  }

// 判断是否在 terminal 控制台中
const _tty = Deno.isTTY();
console.log(_tty);
// export: { stdin: true, stdout: true, stderr: true }
//  stdin: 是否为标准输入
//  stdout: 是否为标准输出
//  stderr: 是否为标准错误


const _pid = Deno.pid;
console.log(_pid);
// export: {number}进程ID
 
```
